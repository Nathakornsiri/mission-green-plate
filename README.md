# mission-green-plate
ทำเว็บแอป 2user มีฝั่งครู และนักเรียน

บนเว็บแอปหน้าแรกคือเลือกบทบาทว่าเป็นครูหรือนักเรียน

หน้าแรกสุดเป็นการเลือกบทบาทและด้านล่างการเลือกบทบาทจะมีการแนะนำตััวเราว่าหน้าเว็บเป็นยังไงใช้ยังไงมีประโยชน์อย่างไร

if : เป็นครู > กรอกรหัสประจำตัวครู > .สามารถดูประวัติส่วนบุคคลของครู ในหน้านี้จะมีชื่อ รูปโปรไฟล์ แท็บห้องที่ประจำชั้น > คลิกที่แท็บห้องที่ประจำชั้น > จะเจอกับหน้ารายชื่อนักเรียนแบบใบรายชื่อเเละมีติ้กว่าใครกินข้าวหมดหรือไม่หมดบ้าง
.สามารถ search ได้ว่าเด็กกินข้าวหมดไหม กินกับอะไร > ช่อง search จะเป็นเลขประจำตัวหรือชื่อนักเรียนก็ได้ และจะมีแจ้งไว้เลยว่ากินอะไรบ้างกินหมดหรือไม่เรียงเป็นของแต่ละวัน

elseif : เป็นนักเรียน > กรอกเลขประจำตัวนักเรียน > .สามารถดู leaderboard ได้ว่าในเกมใครคือที่1 > เรียงลงมาจากบนลงล่างเป็นคะแนนโดยรวมทั้งหมดจากบนมาล่าง
.ดูสถิติการเล่นเกมส่วนตัว่าไปถึงไหนแล้ว > ในนี้จะขึ้นเป็นหน้าด่านเกมว่าไปถึงไหนแล้ว
.ดูประวัติการกินข้าวหมดว่ากินอะไรไปบ้างและกินหมดไหม > ในหน้านี้จะเห็นเมนูที่เคยกินมาแล้ว และ กินหมดหรือไม่ ในแค่ละวัน
.มีช่อง comment ไว้เพื่อเสนอกับข้าวรายวัน > หน้านี้จะคล้ายๆแบบฟอร์มที่มีการกรอกได้ว่าวันนี้อยากกินอะไรอยากได้อะไรนเป็นพิเศษหรือไม่ อยากได้ออะไรเพิ่มหรือปรับปรุงตรงไหนไหม

หน้าตู้จะมี logic คิอ
สแกนNFCของคนนั้นๆ เข้ากับ RFID scan > ข้อมูลของเด็กคนนั้นๆจะส่งไปที่ raspberry pi > raspberry pi จะสั่งให้กล้องทำงานเริ่มตรวจจับการกินเหลือได้ > พอกล้องตรวจเสร็จก็จะส่งค่ากลับมาที่ raspberry pi > raspberry pi จะส่งค่าขึ้นเว็บและโชว์เกมให้เล่นทันทีหากกินหมด
กินไม่หมดมึงอดเล่นไออ้วน
